## how to run it

**requirements**
- Java 8 or higher
- A terminal / command prompt

**steps**
```bash
# clone the repo
git clone https://github.com/v-nne/BMIQ.git

# navigate into the folder
cd BMIQ

# compile
javac *.java

# run
java HealthProgram
```

---

## supported units

both **metric** and **imperial** are supported across all calculators.

| metric | imperial |
|---|---|
| kg, m, cm | lbs, inches |
